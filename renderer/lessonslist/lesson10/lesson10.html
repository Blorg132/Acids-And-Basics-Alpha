Since many common acids & bases are commonly clearless liquids, it would be difficult to differentiate between acids & bases. Therefore, the following methods are used by scientists.

  • Indicators - A substance which undergoes observerable, chemical changes when being added to an acid / base. The resulting color is what determines whether a particular chemical is acidic / basic.
    Ex. When bromothyl-blue is added to an acid, solution becomes yellow, while becomes blue in basic solutions.
  • Litmus Paper - A type of paper that changes color when being submerged in a particular solution. Ex. When litmus turns red : Acid / When litmus turns blue : Base / If litmus does not change color at all : Neutral


The above methods can also be used to describe the pH of the salt that is formed as well.
